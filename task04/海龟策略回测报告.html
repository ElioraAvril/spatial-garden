策略核心逻辑

唐奇安通道

上轨 = 前20日最高价
下轨 = 前20日最底价
中轨 = (上轨+下轨)/2
离场线 = 前10日最底价

ATR头寸管理

单位 = (权益×1%) / (2×ATR)
止损 = 入场价 - 2ATR
加仓 = 每上涨0.5ATR
最大加仓 = 4个单位

入场条件

收盘价突破上轨 → 买入
连续突破上轨 → 无重复信号
(用shift(1)防未来函数)

离场条件

价格跌破离场线 → 平仓
价格跌破止损价 → 止损
移动止损随价格上涨上移

使用心得与注意事项
  1. 震荡市避免使用：横盘行情中假突破频繁，连续止损会快速消耗本金。入场前应先判断市场是否处于趋势状态。
  2. 参数因人而异：20日/55日经典参数不一定最优，建议根据具体品种的波动特征和历史回测结果调整。
  3. 加仓是双刃剑：加仓放大利润的同时也放大亏损。趋势判断正确时收益可观，但判断错误时损失更大。
  4. 心理考验极大：胜率仅25%-30%，连续多次止损后仍要严格执行系统——这是海龟策略最难的部分。
  5. 组合投资：建议同时跟踪多个品种，分散单一品种的假突破风险。
  6. 防止未来函数：通道计算必须用前N日数据(shift(1))，绝不能包含当天数据，否则信号会失效。
  7. 移动止损：价格突破入场价+0.5ATR后，止损上移至当前价-2ATR，实现"让利润奔跑"。
